# README — Coorélation.ipynb

Ce notebook propose une analyse exploratoire, une réduction de dimension (PCA) et une étude des corrélations entre variables chimiques et énergétiques à partir du fichier Excel inital comprenant 3 classses, 6 produits pour l'alimentation animale.

## Objectifs
- Importer et explorer les données
- Visualiser les corrélations (matrices, barres)
- Comprendre les liens entre variables chimiques et énergétiques
- Visualiser les regroupements par classe ou produit, comprendre les relations entre catégories et variables chimiques/énergétiques (ACP projection dans un espace 2D, 3D)

## Fichier utilisé
- `iodaa_2025_fichier_test.xlsx`



## Prérequis
- Python 3
- pandas, numpy, seaborn, matplotlib, scikit-learn, plotly

## Utilisation rapide
1. Placez le fichier Excel dans le dossier du notebook
2. Installez les bibliothèques si besoin :
   ```bash
   pip install pandas numpy seaborn matplotlib scikit-learn plotly openpyxl
   ```
3. Exécutez le notebook
